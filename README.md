# 🛡️ VeriTrade AI
**AI-Powered Compliance & ESG Risk Assessment for Global Trade**

VeriTrade AI is a high-fidelity compliance engine designed to screen shipping manifests for international sanctions, conflict minerals, and ESG (Environmental, Social, and Governance) violations. By combining **deterministic rule-based logic** with **generative AI reasoning**, it provides an immutable audit trail secured by cryptographic hashing.

---

## 🚀 Key Features
* **Hybrid AI Engine:** Uses hardcoded compliance rules for instant validation and **Noah AI (LLM)** for deep ethical and geopolitical reasoning.
* **ESG Verification:** Automatically checks for certifications like ISO-14001, Carbon Neutrality, and Social Accountability (SA8000).
* **Cryptographic Fingerprinting:** Generates a **SHA-256 hash** for every audit record, ensuring data integrity and blockchain readiness.
* **Strategic Insights:** Provides a narrative analysis of trade risks that simple logic-based systems miss.
* **Modern Dashboard:** A sleek, dark-themed UI built for high-stakes trade monitoring.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Backend:** FastAPI (Python 3.10+)
* **AI:** Noah AI API
* **Security:** SHA-256 Hashing
